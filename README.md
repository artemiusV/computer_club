### Установка зависимотсей

Для установки и запуска проекта, необходим NodeJS v20+

##### Для установки зависимостей, выполните команду:

```
npm start i
```

##### Чтобы запустить сервер для разработки, выполните команду:

```
npm start
```

##### Версия NodeJS

v20.11.1

##### Версия npm

v10.2.4
